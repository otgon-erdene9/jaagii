# ⚡ QuizBlast — Kahoot шиг үнэгүй quiz тоглоом

## 🚀 Хэрхэн ажиллуулах

1. `index.html` файлыг browser-т нээнэ үү (давхар click)
2. Тоглоом шууд ажиллана — интернэт шаардлагагүй!

---

## 🎮 Тоглоомын горимууд

### 🎯 Хост болох (Тоглоом зохион байгуулах)
1. **"Хост болох"** товч дарна
2. Quiz сонгоно уу
3. **PIN код** гарч ирнэ — найзуудаа урина
4. Найзууд нэгдсэний дараа **"Тоглоом эхлүүлэх"** дарна

### 🎮 Тоглогч болох
1. **"Тоглох"** товч дарна
2. **PIN код** болон **нэрээ** оруулна
3. Хост тоглоом эхлүүлэхийг хүлээнэ
4. Асуулт гарахад хурдан хариулна — хурдан = илүү оноо!

### ✏️ Шинэ Quiz үүсгэх
1. **"Quiz үүсгэх"** товч дарна
2. Quiz нэр оруулна
3. Асуулт, хариулт нэмнэ
4. Зөв хариултыг тэмдэглэнэ
5. **"Хадгалах"** дарна

## 💡 Нэг компьютер дээр тоглох заавар

1. `index.html`-ийг нэг tab-д нээнэ — **Хост болно**
2. Өөр tab нээж — **Тоглогч болно** (PIN код оруулна)
3. 3, 4 tab нээж олон тоглогч нэмж болно!
